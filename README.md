# cedar: Cost-Effectiveness Decision Analysis in R

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

## Overview

The **cedar** package provides tools and functions for conducting cost-effectiveness analysis (CEA). It supports decision-analytic modeling, probabilistic sensitivity analysis, and value of information calculations commonly used in health economics and policy evaluation.

## Installation

You can install the development version of cedar from GitHub:

```r
# install.packages("devtools")
devtools::install_github("username/cedar")
```

## Features

- Decision-analytic modeling framework
- Probabilistic sensitivity analysis tools
- Value of information calculations
- Cost-effectiveness analysis utilities
- Support for common health economic evaluation methods
